# MySSConnect

## 機能
* 毎日更新のSSが見れる
* tagとtitleで任意のSSを検索できる
* お気に入りをすることができる
* 好きなブログを優先させることが出来る
## 特徴
* タイトルの重複を減らしている(同じタイトルのものを同じstoryとして扱う)
* 各storyにtagをつけている(例：けいおん!)
